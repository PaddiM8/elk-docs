# Iteration (iter)
## all(list: List)

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| list      | List |             |

### Returns

(Boolean) Whether or not all the values in the list evaluate to true.

## any(list: List)

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| list      | List |             |

### Returns

(Boolean) Whether or not one of the values in the list evaluates to true.
